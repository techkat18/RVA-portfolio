# GEC ELEC 6: Reading Visual Arts E-Portfolio

Welcome to my interactive e-portfolio for **GEC ELEC 6 (Reading Visual Arts)**! This project is a hand-coded, single-page application (SPA) designed to showcase my academic journey, visual art analyses, and personal reflections. 

As a BSIT student, this project bridges the logic of programming with the beauty of visual arts—proving that code is just another canvas.

## Features

* **Immersive "Desk" UI (Parallax):** A custom physics-based parallax effect tied to mouse movement makes the desk elements (coffee cups, pencils, polaroids) float and shift dynamically.
* **Flawless CSS Grid Layout:** A responsive, zero-overlap grid system that intricately weaves text cards and floating character illustrations side-by-side.
* **Dynamic Theme Toggle:** A custom switch in the navigation bar that seamlessly transitions the entire website's color palette between a dreamy Royal Purple and an elegant Maroon.
* **Interactive Modals:** Custom-built pop-ups (lightboxes) that allow users to view embedded PDF discussion papers and full-resolution scanned exam images without ever leaving the page.
* **Playful Micro-interactions:** * A custom trailing CSS cursor (a purple/gold dot).
  * 3D card-tilt hover effects.
  * A highly interactive 5-star rating widget (which cleverly rejects any rating lower than 5 stars with a shaking animation!).
* **Pure CSS & SVG Graphics:** To keep the site lightweight and crisp, all UI icons and decorative elements (like the top-down coffee cup and pencil) were coded using pure CSS and inline SVGs.

## What's Inside

The portfolio is divided into 5 seamless chapters:
1. **Photography:** Explores the technicalities of the exposure triangle, the language of photography, and the ethical dimensions of visual storytelling.
2. **Docu Reflection:** A deep dive into Joey Velasco's *Sa Kambas ng Lipunan*, reflecting on art as a medium for social commentary and compassion.
3. **Assessments:** "Artifacts of Learning" showcasing digitized quizzes and midterm examinations, complete with highlighted Q&As and image-viewing modals.
4. **My Experience:** A personal reflection comparing the rigid structure of coding to the emotional fluidity of painting.
5. **Feedback:** Course evaluation, personal takeaways, and a fun interactive rating system.

## Tech Stack

This project was built entirely from scratch without any external frameworks or libraries:
* **HTML5:** Semantic structuring and custom data attributes.
* **CSS3:** Advanced CSS Grid, Flexbox, Keyframe Animations, CSS Variables (Custom Properties), and 3D Transforms.
* **Vanilla JavaScript:** Intersection Observers for scroll-reveals, mouse event listeners for parallax/tilt effects, and DOM manipulation for modals and theme toggling.
